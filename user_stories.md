# User Stories Backlog

## Inventory Management

### US1
As a user  
I want to add an item  
So that I can track my belongings

Acceptance Criteria:

- user enters item name
- user enters purchase date
- item appears in inventory

---

### US2
As a user  
I want to edit item details  
So that I can update information

Acceptance Criteria:

- user can change name
- user can update category

---

### US3

So that I can remove unused entries
